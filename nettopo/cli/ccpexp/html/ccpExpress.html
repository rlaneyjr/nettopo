The Cisco Configuration Professional Express (Cisco CP Express) is an embedded, device-management tool that provides the ability to bootstrap and provision an Integrated Services Router (ISR).

Security features will be supported only if the IOS version is 15.5(1)T and above and security license is enabled.

Cisco Configuration

Professional Express 3.2

  • Basic Settings

    Configure the device hostname, domain name, Timezone, NTP, DNS server, IPv4 DHCP Pools and DNS Proxy.

  • Interface and Connections

    Configure all device interfaces including LAN and WAN interfaces. Setup DSL, Ethernet, 3G or 4G WAN links or create Vlans and Loopback interfaces to configure interface attributes.

  • Quick Setup Wizard

    This wizard helps to bring up your WAN/LAN connectivity quickly. If the router is not a factory fresh using quick setup wizard is not recommended because using that will lead to configuration issues.

  •  
    Dashboard

    View basic diagnostic information like Version, Interfaces, Software details, Flash/CPU utilization stats etc. Security and Application Visibility & Control info is listed based on device support.

  • Identity

    Configure new Users on the device with specified privilege levels and manage the End User View.

  • Static Routing

    Configure IPv4 and IPv6 static routes.

  • Cisco Active Advisor

    Send hardware and software product usage information to Cisco.

  • Any cli to the box

    Configure IOS cli commands, run show commands.

  •  
    Troubleshoot

    Troubleshoot reachability to other IPv4 or IPv6 destinations using Ping or Traceroute utilities.

  •  
    Security

    Comprehensive solution that includes key components of threat defense with firewalling, intrusion prevention, VPN, and content security functions.

  • Wi-Fi

    Configure the wlan access point to create new SSIDs and authentication keys.

  • ACL

    Configure the ACL.
